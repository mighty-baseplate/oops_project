Streams API Demo

This analytics page demonstrates Java Streams API features:

  • filter() - Filtering passed students (score >= 50)
  • mapToInt() and average() - Calculating average scores
  • count() - Counting total students
  • groupingBy() - Grouping students by exam type (see console)
Note: For actual data, students need to submit exams first. Currently showing demo visualization with sample data.
